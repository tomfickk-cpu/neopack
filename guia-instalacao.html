▸ GUIA DE INSTALAÇÃO

Siga os passos abaixo para instalar o NeoPack no seu computador

🪟 Windows 🐧 Linux 🍎 macOS 🐳 Docker
PRÉ-REQUISITOS

📋 O que você precisa ter instalado

  • Node.js 18+ — nodejs.org (recomendado: versão 20 LTS)
  • Python 3.10+ — python.org
  • Git 2.30+ — git-scm.com
  • Go 1.21+ (opcional, apenas para agentes Go) — go.dev
💡 Dica: Para verificar o que já tem instalado, abra o terminal e digite:
node --version
python --version
git --version
Se aparecer a versão, está OK. Se não, instale o que faltar.
PASSO 1

📦 Extraia o pacote

Após o download, localize o arquivo NeoPack-PRO-v3.0.0.zip (ou Enterprise).

# Clique com o botão direito no ZIP → "Extrair tudo"
# Escolha uma pasta de destino (ex: C:\NeoPack)

A estrutura extraída será:

C:\NeoPack\
├── scripts\ # Scripts de instalação
├── config\ # Arquivos de configuração
├── .opencode\agents\ # Arquivos dos 48 agentes
├── README.md # Instruções
└── LICENSE.txt # Sua licença
PASSO 2

⚡ Execute o setup (1 clique)

Clique com o botão direito no script setup.ps1 e escolha "Executar com PowerShell" (como Administrador).

Ou manualmente:

cd C:\NeoPack\scripts
powershell -ExecutionPolicy Bypass -File setup.ps1
⚠️ Importante: Execute como Administrador. O script vai instalar pacotes npm globais e precisa de permissão.

O que o script faz automaticamente:

  • Verifica Node.js, Python e Git
  • Instala o OpenCode CLI globalmente
  • Instala todos os 12 MCP Servers (Puppeteer, Playwright, GitHub, Brave Search, PostgreSQL, Redis, Figma, Notion, etc.)
  • Instala os 7 LSPs (TypeScript, HTML, CSS, JSON, Python, Markdown, Go)
  • Instala os 3 formatters (Prettier, Ruff, Gofmt)
  • Cria o projeto ~/NeoPack-PRO com todos os 48 agentes configurados
PASSO 3

🔑 Configure sua chave de API

O OpenCode precisa de uma chave de API para funcionar. Você pode usar o modelo gratuito (DeepSeek) ou modelos pagos.

# Abra o terminal e digite:
opencode auth login
✓ Autenticado com sucesso!

O modelo padrão já é o DeepSeek V4 Flash (gratuito). Se quiser models premium (Go), configure manualmente.

PASSO 4

🚀 Pronto! Use o NeoPack

Navegue até a pasta do projeto e inicie o OpenCode:

cd ~\NeoPack-PRO
opencode

Agora você tem 48 agentes especialistas disponíveis. Teste alguns comandos:

# Exemplos de uso:
/agent rust-dev # Agente Rust
/agent docker-agent # Agente Docker/K8s
/agent postgres-agent # Agente PostgreSQL
/agent security-audit # Agente de Segurança
/agent mlops-agent # Agente MLOps
/agent web3-agent # Agente Web3/Blockchain

ALTERNATIVA

🐳 Usando Docker

O NeoPack já vem com Dockerfile pronto. Use em qualquer ambiente com Docker:

# Build da imagem
docker build -t neopack .

# Execute
docker run -it --rm neopack
🔧 SOLUÇÃO DE PROBLEMAS

Problemas comuns e soluções

  • "npm não é reconhecido": Instale o Node.js em nodejs.org e reinicie o terminal
  • "ExecutionPolicy" bloqueado: Execute: Set-ExecutionPolicy Unrestricted -Scope CurrentUser
  • "python não encontrado": Instale Python em python.org e marque "Add to PATH"
  • "Permission denied" (Linux/Mac): Use sudo antes do comando
  • MCP server não conecta: O script instala todos globalmente. Se falhar, instale manualmente com npm install -g @anthropic/mcp-server-puppeteer
